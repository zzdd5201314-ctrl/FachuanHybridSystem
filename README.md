# 法穿AI案件管理系统V26.34.0

当一位律师接手一个新案件时，他要面对的是什么？客户递过来一堆散乱的材料：身份证、合同、转账截图、聊天记录、营业执照……然后是漫长的整理、录入、起草、递交、等待、再整理，周而复始。大部分时间消耗在事务性工作上，真正用于专业判断和策略思考的时间被严重挤压。

**法穿AI存在的意义，就是把这条完整的工作链变成一条自动运转的流水线。**

## 从接洽客户开始

潜在客户第一次联系，法穿AI便启动全流程服务。**身份证件OCR自动识别**，姓名、证件号、地址一字不错地填入档案。企业客户只需输入公司名称或上传营业执照，**企业信息自动回填**：法定代表人、注册资本、注册地址等全部结构化入库。财产线索同步登记，随时可查。

这些数据会成为后续所有文书的**唯一数据源**：输入一次，全局复用。

## 签约与指派

谈妥委托事宜后，系统根据当事人信息和案件情况，**自动生成完整的委托合同文本**：条款、费用约定、代理权限等全部基于模板智能填充。补充协议同样支持自动起草生成。

合同签订后可一键同步至律所内部OA系统，**案件编号自动关联**，内外数据打通，无需人工两边对账。

## 立案材料准备与智能文书生成

这是传统模式下最耗时的环节：律师要逐一甄别分类、扫描排版、起草文书、组卷归档，动辄数天。

在法穿AI中：律师将所有电子材料一次性上传，系统基于AI**智能识别材料类型并自动分类归档**（身份证明类、合同协议类、财务凭证类、通讯记录类），同时根据已识别的信息**全套同步生成完整立案文书**：起诉状、答辩状、授权委托书、律师事务所函、送达地址确认书、财产保全申请书、证据清单、退费账户确认书等一应俱全，每份都根据案件信息自动填充，直接落入正确位置拿来即用。

诉讼过程中也可随时针对单个文件重新生成，**自动覆盖原位置**，无需手动挪动。从散乱材料到完整立案包，时间从数天压缩到以小时计。

## 一键双通道提交立案

传统做法：先在律所内部管理系统走审批，审批通过后再登录法院"一张网"手动填写一遍几乎相同的信息，重复劳动。

法穿AI的做法：**同时驱动两条通道并行完成。**

目前已覆盖**大成、德恒、锦天城、泰和泰、金诚同达、中银、隆安**等一系列采用IMS系统的律所：一边自动填写IMS立案表单、上传材料、提交审批；另一边同时登录法院一张网平台、填写立案信息、上传电子材料。两条通道**并排推进**，互不干扰，一次操作双向完成。

## 案件进入诉讼程序后的全自动陪伴

### 法院短信自动处理链路

诉讼过程中法院会不断发送短信通知：立案通知书送达了、开庭传票出了、举证期限到了、判决书下来了……每一条都意味着律师要打开链接、下载PDF、归档、更新日志、判断是否需要行动、设置日程提醒。

在法穿AI中，这条长长的操作链被压缩成一个动作：**转发短信。**

律师收到法院短信后通过iOS快捷指令或安卓SMS Forwarder转发给法穿AI，之后一切自动完成：智能解析案号/当事人/案件类型 → 自动下载文书PDF → 匹配已有案件 → 规范重命名 → 自动归档到对应文件夹 → 校验诉讼费是否异常 → 飞书/Telegram实时通知律师。整个过程无人值守，7×24小时运行。

### 证据持续补充与管理

新证据陆续产生？继续上传即可：AI分类引擎自动识别类型并归入已有文件夹体系。聊天记录取证支持**视频截帧+OCR文字提取**，一键生成证据文档。

### 庭审准备支持

临近开庭，系统帮助快速梳理案件全貌：AI辅助生成**庭审问题清单**和**庭审提纲**，确保庭前准备无遗漏。更强大的是**多Agent模拟庭审功能**：多个AI角色分别扮演原告、被告、法官进行对抗式推演，律师可代替任意角色发言，提前预判对方论点和法官关注点。

### 财务工具随需调用

内置专业**金融计算器**，支持利息、LPR分段、违约金等多种复杂场景，结果直接用于文书引用。快递物流查询能力可追踪**邮寄送达状态**：上传邮单图片或输入运单号，自动查询顺丰/EMS轨迹并导出PDF报告作为送达凭证。

### 信息中枢：永不遗漏的消息管家

除了法院短信，法穿AI的**信息中转站**统一聚合IMAP邮箱和YZW收件箱的所有消息，支持发件人白名单/黑名单过滤，重要附件按需下载预览。一个入口掌控全部信息动态。

### 开放能力：与AI Agent无缝协同

通过MCP（Model Context Protocol），任何AI Agent都可以用自然语言直接调用系统全部能力：查询案件进展、检索客户信息、获取合同数据……200余个API接口全面开放，让法穿AI成为律所智能化转型的技术底座。

## 结案与归档

判决书或调解书到达后，同样通过法院短信自动处理链路完成接收、下载、归档。此时案件文件夹已经是一套**完整的、有序的、可直接交付的案件卷宗**：从立案材料到结案文书，每一个环节有迹可循。合同收费状态同步更新，财务数据自动结算。
